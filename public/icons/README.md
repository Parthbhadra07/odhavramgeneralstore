# PWA Icons

Add the following PNG files for PWA install support:

- `icon-192.png` (192×192)
- `icon-512.png` (512×512)

You can generate these from your logo using [realfavicongenerator.net](https://realfavicongenerator.net/) or:

```bash
npx @capacitor/assets generate --iconBackgroundColor "#16a34a"
```
